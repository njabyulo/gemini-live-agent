# Deployment Guides

## Scope

Deployment guides for the three app surfaces in this repo.

## Guides

- `web.md`
- `api.md`
- `agent-live.md`

## Runtime Split

1. `apps/web` is hosted on Cloudflare
2. `apps/api` is hosted on Cloudflare
3. `apps/agent-live` is hosted on Google Cloud Run
4. The live agent itself remains clearly hosted on Google Cloud for judging
